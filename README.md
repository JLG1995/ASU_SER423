# ASU Mobile App Development (SER 423)

This repository showcases a collection of assignments and small projects I completed for **SER 423: Mobile Systems** at **Arizona State University**, taught by **Dr. Tyler Baron**.  
The course focused on **React Native** development, emphasizing mobile architecture, lifecycles, user interfaces, networking, and integration with native device features.

---

## Overview

**Course:** SER 423 – Mobile Systems  
**Institution:** Arizona State University  
**Instructor:** Dr. Tyler Baron  
**Technologies Used:** JavaScript, React Native, Expo, Node.js  
**Textbook:** *React Native Cookbook (2nd Edition)* by Dan Ward  

---

## What I Learned

- Designing and building mobile user interfaces in React Native  
- Handling component lifecycles and navigation  
- Using asynchronous functions and remote APIs  
- Managing local and cloud data storage  
- Incorporating maps, location services, and media  
- Applying performance optimization techniques for mobile apps  

Each module assignment reinforced real-world concepts of mobile software design, emphasizing both **technical correctness** and **user experience**.
